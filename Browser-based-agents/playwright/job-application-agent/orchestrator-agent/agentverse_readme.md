# Job Application Agent

Your conversational assistant for managing job applications end-to-end.

## What it does

**Profile management**
- Upload a resume (PDF, DOCX, or TXT) directly in chat — it is parsed and stored.
- Edit your profile in plain English: name, contact details, work authorisation,
  EEO fields, education, experience, and more.
- Say `show my profile` or `whoami` at any time to review what is stored.

**Applying for jobs**
- Paste any Greenhouse job posting URL to start an application.
- The agent fills the form automatically using your stored profile and resume,
  streams live progress, and asks you only for answers it cannot infer.
- **A one-time payment is required per application submission.** You will see a
  payment card before the form fill begins — approve it to proceed.
- Every answer you give is saved back to your profile so future applications
  need less input.

## Example messages

- `hi` — onboarding walkthrough
- `show my profile`
- `my phone is +1-555-1234`
- `set my work auth to US Citizen`
- `https://boards.greenhouse.io/company/jobs/12345` — start an application
