# OpenClaw examples

Examples that integrate **[OpenClaw](https://openclaw.ai)** with the Fetch.ai stack (Agentverse, uAgents, ASI:One).

| Example | Description |
|--------|-------------|
| [`fetchai-openclaw-orchestrator/`](fetchai-openclaw-orchestrator/) | Local **connector** + **orchestrator** agent — signed task plans, repo health analyzer, weekly dev report workflow. |
| [`agentverse-caller/`](agentverse-caller/) | OpenClaw **skill** — search the Agentverse marketplace and message agents by shortcut, name, or address. |

Install and run instructions are inside each folder’s `README.md` / `SKILL.md`.
